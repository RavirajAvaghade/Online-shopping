# 🛒 Online Shopping System

A dynamic and user-friendly **Online Shopping Web Application** developed using Java technologies. This project allows users to browse products, add them to cart, and purchase items seamlessly.

---

## 🚀 Features

* 👤 User Registration & Login
* 🛍️ Browse Products
* 🛒 Add to Cart
* 💳 Buy Products
* 📦 Order Management
* 🔍 Product Details View

---

## 🛠️ Technologies Used

* **Core Java**
* **Advanced Java (JDBC, Servlets, JSP)**
* **SQL (Database Management)**
* **HTML, CSS (Frontend UI)**

---

## ⚙️ Tools & Environment

* Apache Tomcat Server
* MySQL Database
* Eclipse / VS Code

---

## 📌 Project Functionality

* Users can register and log in securely
* Products are fetched from the database using JDBC
* Users can add products to cart and place orders
* Backend is handled using Servlets and JSP for dynamic content

---

## 💡 What I Learned

* Java Web Development concepts
* Database connectivity using JDBC
* MVC architecture basics
* Handling user sessions and requests

---

## ▶️ How to Run

1. Clone the repository
2. Import project into Eclipse / VS Code
3. Configure Apache Tomcat Server
4. Setup MySQL database and update DB credentials
5. Run the project on server
